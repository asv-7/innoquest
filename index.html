      • 
                    
    • Test with Examples: Run your code with different inputs to find edge cases
      • 
                    
    • Use Loops Carefully: Off-by-one errors are common in loops (i < vs i <=)
      • 
                    
    • Watch Data Types: Make sure you're comparing the right data types (5 vs "5")
